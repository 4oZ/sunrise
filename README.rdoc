= Sunrise CMS

Aimbulance mini content management system for programmers.

== Instructions

Create new project

  rails new [PROJECT_NAME] -m https://gist.github.com/1523718.txt


== Export data

=== XML, JSON, CSV

  GET /manage/users/export.xml
  GET /manage/users/export.json
  GET /manage/users/export.csv

=== Excel

  gem "ruby2xlsx", "~> 0.0.1"

  GET /manage/users/export.xlsx
